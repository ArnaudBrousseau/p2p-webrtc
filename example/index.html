Minimal WebRTC console based chat demo

Everything happens in your web console. So open it.

  • Alice calls createChannel(). This will generate an offer description.
  • Bob (a separate browser/tab) loads the same page. Call joinChannel(offerDescription). This will generate an answerDescription that Bob must send back to Alice
  • Alice call ackPeerConnection(answerDescription) to open the data channel.
  • From that point on, Alice and Bob can send each other messages through sendMessage(message)
